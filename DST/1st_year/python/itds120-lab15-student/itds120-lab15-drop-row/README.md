# Row Score — Remove the Row with the Lowest Sum (Advanced)

จงเขียนโปรแกรมที่อ่านข้อมูล **matrix (2D array)** จากผู้ใช้งาน แล้ว **ลบแถวที่มีผลรวมของค่าต่ำที่สุด** ออกจาก matrix

**ข้อกำหนด:**

- ต้องสร้าง **matrix ใหม่** เพื่อเก็บผลลัพธ์ที่ได้จากการลบแถว (ห้ามแก้ไข matrix เดิมโดยตรง)  
- ถ้ามีหลายแถวที่มีผลรวมต่ำที่สุด ให้ลบ **แถวที่พบก่อน (index ต่ำสุด)** เท่านั้น  
- ต้องใช้ **NumPy** ในการคำนวณ  
- มีคำใบ้อยู่ใน `main.py`

## ข้อมูลนำเข้า (Input)

ผู้ใช้งานจะป้อนข้อมูล 2 ค่าแรกคือ **จำนวนแถว (`row`)** และ **จำนวนหลัก (`column`)**  จากนั้นจะป้อนค่าของ matrix ทั้งหมดตามลำดับ  

**รูปแบบข้อมูลนำเข้า**

```text
<จำนวนแถว> <จำนวนหลัก>
<แถวที่ 1>
<แถวที่ 2>
...
<แถวที่ n>
```

**ตัวอย่าง**

```text
3 4
10 20 30 40
5 5 5 5
1 2 3 4
```

## ผลลัพธ์ (Output)

โปรแกรมต้องแสดงผล matrix ที่ได้หลังจากลบแถวที่มีผลรวมของค่าน้อยที่สุดออกไป โดย print matrix จาก NumPy ออกมาเลย

**Expected Output:**

```text
[[10. 20. 30. 40.]
 [ 5.  5.  5.  5.]]
```
