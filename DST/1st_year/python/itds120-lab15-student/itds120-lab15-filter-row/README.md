# Row Filter — Keep Rows with Row-Max >= Threshold

ให้เขียนโปรแกรมเลือกเฉพาะ "แถว" ของเมทริกซ์ที่มี **ค่ามากที่สุดของแถวนั้น >= ธรณี (threshold)** แล้วแสดงเมทริกซ์ที่เหลือ

**ข้อกำหนด:**

- ใช้: `np.max()` ที่ระบุ axis ในการหา max ของแต่ละแถว
- ธรณี (threshold) รับจากผู้ใช้ (integer)

## ข้อมูลนำเข้า (Input)

1. ผู้ใช้งานจะป้อนข้อมูล 2 ค่าแรกคือ **จำนวนแถว (`row`)** และ **จำนวนหลัก (`column`)**
2. จากนั้นจะป้อนค่าของ matrix ทั้งหมดตามลำดับ
3. สุดท้ายผู้ใช้จะป้อนค่า **ธรณี (threshold)**

**รูปแบบข้อมูลนำเข้า**

```text
<จำนวนแถว> <จำนวนหลัก>
<แถวที่ 1>
<แถวที่ 2>
...
<แถวที่ n>
<threshold>
```

**ตัวอย่าง**

```text
3 4
10 20 30 40
11 12 13 14
1 2 3 4
15
```

## ผลลัพธ์ (Output)

โปรแกรมต้องแสดงผล matrix ที่ได้หลังจากลบแถวที่มีค่ามากที่สุดของแถวนั้นน้อยกว่า threshold ออกไป โดย print matrix จาก NumPy ออกมาเลย

**Expected Output:**

```text
[[10 20 30 40]]
```
