## Calculate the BMI

จงเขียนโปรแกรมเพื่อคำนวณ BMI จากค่า input: น้ำหนักในหน่วยกิโลกรัม และส่วนสูงในหน่วย เซนติเมตร และแสดงผลลัพธ์ของค่าน้ำหนักดังตาราง 

|       BMI      | Weight Status |
|:--------------:|:-------------:|
|   น้อยกว่า 18.5  |  Underweight  |
|  18.5 ถึง 24.9  |     Normal    |
|   25.0 ถึง 29.9 |   Overweight  |
|    มากกว่า 30   |     Obese     |

**หมายเหตุ:** นักศึกษาสามารถสมมติได้ว่าผู้ใช้จะ input เป็นตัวเลขเท่านั้น ไม่มีตัวอักษร 
<hr>

**ตัวอย่างที่ 1:**

**Input:** น้ำหนัก 52 กิโลกรัม ส่วนสูง 170 เซนติเมตรตามลำดับ
```
52
170
```
**Expected output:** หลังจากคำนวณแล้ว โปรแกรมควรแสดงผลลัพธ์เป็น
```
Underweight
```
<hr>

**ตัวอย่างที่ 2:**

**Input:** น้ำหนัก 70 กิโลกรัม ส่วนสูง 155 เซนติเมตรตามลำดับ
```
70
155
```
**Expected output:** หลังจากได้รับชื่อแล้ว โปรแกรมควรแสดงผลลัพธ์เป็น 
```
Overweight
```
