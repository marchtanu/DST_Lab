## Grade Calculation

จงเขียนโปรแกรมคำนวณเกรดของวิชา ITDS120 Fundamentals of Programming
ตามรายละเอียดของคะแนนในหมวดต่าง ๆ ดังนี้

|    Evaluation    | คะแนนเต็ม | คิดเป็นร้อยละ |
|:----------------:|:--------:|:----------:|
| Lab              |    12    |     10     |
| Assignment       |    12    |     10     |
| Quiz             |    12    |     10     |
| Check Point Exam |    80    |     50     |
| Project          |    100   |     20     |

โดยเกรดจะคำนวณจากผลรวมของคะแนนแต่ละส่วนถ่วงน้ำหนักตามสัดส่วน จากนั้นนำคะแนนสุทธิที่ได้ไปคำนวณ เกรดตามเกณฑ์ต่อไปนี้

| เกรด |  ช่วงคะแนน |
|:----:|:---------:|
|   A  |   85-100  |
|  B+  |  80-84.9  |
|   B  |  75-79.9  |
|  C+  |  70-74.9  |
|   C  |  65-69.9  |
|  D+  |  60-64.9  |
|   D  |  50-59.9  |
|   F  | น้อยกว่า 50 |

โปรแกรมจะรับ input เป็นคะแนนในแต่ละหมวดตามลำดับ คือ Lab, Assignment, Quiz, Check Point Exam, และ Project หลังจากคำนวณแล้ว โปรแกรมควรแสดงผลลัพธ์เป็นเกรดตามเกณฑ์ที่ได้

**หมายเหตุ:** นักศึกษาสามารถสมมติได้ว่าผู้ใช้จะ input เป็นตัวเลขเท่านั้น ไม่มีตัวอักษร 
<hr>

**ตัวอย่างที่ 1:**

**Input:** คะแนนจะถูกใส่มาเป็น Lab, Assignment, Quiz, Check Point Exam, และ Project ตามลำดับ
```
8
8
7
60
80
```
**Expected output:** หลังจากคำนวณแล้ว คะแนนรวมเต็ม 100 จะอยู่ประมาณ 72.67 โปรแกรมควรแสดงผลลัพธ์เป็น
```
Grade: C+
```
<hr>

**ตัวอย่างที่ 2:**

**Input:** คะแนนจะถูกใส่มาเป็น Lab, Assignment, Quiz, Check Point Exam, และ Project ตามลำดับ
```
10
8
12
70
88
```
**Expected output:** หลังจากคำนวณแล้ว คะแนนรวมเต็ม 100 จะอยู่ประมาณ 86.35
โปรแกรมควรแสดงผลลัพธ์เป็น 
```
Grade: A
```
