## Fizz Buzz
จงเขียนโปรแกรมแสดงผล Fizz Buzz ตั้งแต่ 1 ถึง `n` (ช่วง `[1,n]`) โดยโปรแกรมรับ 1 input คือ จำนวนเต็มบวก `n`

หากผู้ใช้ใส่จำนวนไม่ถูกต้อง เช่น จำนวน 0 หรือจำนวนเต็มลบ โปรแกรมจะวนรับค่า input ใหม่จนกว่าจะได้จำนวนเต็มบวก 

จากนั้นโปรแกรมจะแสดงผล Fizz Buzz ตามเงื่อนไข ดังนี้
- หากตัวเลขน้ัน ๆ หาร 3 ลงตัวเท่านั้น เช่น 3, 6
โปรแกรมจะแสดงผลลัพธ์ว่า `Fizz` 
- หากตัวเลขนั้น ๆ หาร 5 ลงตัวเท่านั้น เช่น 5, 10 โปรแกรมจะแสดงผลลัพธ์ว่า `Buzz` 
- แต่ถ้าหากหารท้ัง 3 และ 5 ลงตัวทั้งคู่ เช่น 15, 30 โปรแกรมจะแสดงผลลัพธ์ว่า `Fizz Buzz` 
- ในกรณีหาร 3 หรือ 5 ไม่ลงตัว โปรแกรมจะแสดงตัวเลขน้ัน ๆ



<hr>

**ตัวอย่างที่ 1:**

**Input:** ผู้ใช้ ใส่ตัวเลข `n=15`
```
15
```
**Expected output:** โปรแกรมผลลัพธ์ตามเงื่อนไข ใน ช่วง [1, 15] ตามเงื่อนไข ดังนี้
```
1 2 Fizz 4 Buzz Fizz 7 8 Fizz Buzz 11 Fizz 13 14 Fizz Buzz 
```
<hr>

**ตัวอย่างที่ 2:**

**Input:** ผู้ใช้ใส่ค่าตัวเลข 0, -1, -3, 30 ตามลำดับ 
เนื่องจาก 0, -1 และ -3 ไม่ได้เป็นจำนวนเต็มบวก โปรแกรมจึงวนรับค่าไปเรื่อย ๆ จน เมื่อ `n` มีค่าเป็นจำนวนเต็มบวก คือ `n=30`

```
0
-1
-3
30
```
**Expected output:** โปรแกรมผลลัพธ์ตามเงื่อนไข ใน ช่วง [1, 30] ตามเงื่อนไข ดังนี้
```
1 2 Fizz 4 Buzz Fizz 7 8 Fizz Buzz 11 Fizz 13 14 Fizz Buzz 16 17 Fizz 19 Buzz Fizz 22 23 Fizz Buzz 26 Fizz 28 29 Fizz Buzz 
```
