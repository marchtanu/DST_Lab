## Print a Pattern

จงเขียนโปรแกรมที่รับตัวเลขจำนวนเต็มบวกคี่ `n` ที่มากว่า 1 จากนั้นโปรแกรมจะแสดงผลรูปแบบดังตัวอย่างตามจำนวนแถว `n` 

โปรแกรมจะรับค่า `n` ที่เป็นจำนวนเต็มบวกคี่่เท่านั้น หากผู้ใช้ใส่ `n` ที่ไม่ใช่จำนวนเต็มบวกคี่
ที่มากกว่า 1 เช่น 0 หรือ 1 หรือ จำนวนเต็มลบ หรือ จำนวนเต็มบวกคู่ โปรแกรมจะวนรับค่า input ไปเรื่อย ๆ จนกว่าจะได้จะ `n` ที่เป็นจำนวนเต็มบวกคี่

เมื่อได้ `n` แล้ว โปรแกรมจะแสดงผลลัพธ์เป็นสี่เหลี่ยมที่มีรูปแบบ โดยใช้เครื่องหมาย `*` และ `-` ดังนี้

ตัวอย่าง `n=7` 
```
* - - - - - * 
* * - - - * * 
* - * - * - * 
* - - * - - * 
* - * - * - * 
* * - - - * * 
* - - - - - *
```
<hr>

**ตัวอย่างที่ 1:**

**Input:**
ในกรณีที่ input ไม่ตรงตามเงื่อนไข โปรแกรมจะวนรับค่า ไปเรื่อย ๆ จนกว่าจะเป็นจำนวนเต็มคี่บวก คือ 5
```
0
-4
42
1
5
```

**Expected output:** หลังจากโปรแกรมรับค่า `n` แล้ว โปรแกรมจะแสดงผลลัพธ์ดังต่อไปนี้
```
* - - - * 
* * - * * 
* - * - * 
* * - * * 
* - - - * 
```
