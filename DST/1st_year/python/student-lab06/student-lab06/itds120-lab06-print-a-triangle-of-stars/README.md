## Print a Triangle of Stars

จงเขียนโปรแกรมที่รับตัวเลขจำนวนเต็มบวก `m` โดย `m` กำหนดจำนวนแถว จากนั้นแสดงผลลัพธ์เป็นรูปตามตัวอย่างต่อไปนี้

**หมายเหตุ:** 
นักศึกษาสามารถสมมติได้ว่าผู้ใช้จะ input ตัวเลขจำนวนเต็มบวกเท่านั้น 

<hr>

**ตัวอย่างที่ 1:**
**Input:** ผู้ใช้ input `m=5` 
```
5
```
**Expected output:** โปรแกรมจะแสดงผลลัพธ์ดังต่อไปนี้
```
----*----
---***---
--*****--
-*******-
*********
```
<hr>

**ตัวอย่างที่ 2:**
**Input:** ผู้ใช้ input `m=2`
```
2
```
**Expected output:**  โปรแกรมจะแสดงผลลัพธ์ดังต่อไปนี้
```
-*-
***
```
<hr>

**ตัวอย่างที่ 3:**
**Input:** ผู้ใช้ input `m=10` 
```
10
```
**Expected output:** โปรแกรมจะแสดงผลลัพธ์ดังต่อไปนี้
```
---------*---------
--------***--------
-------*****-------
------*******------
-----*********-----
----***********----
---*************---
--***************--
-*****************-
*******************
```
